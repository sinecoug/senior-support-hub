# Senior Support Hub

Practical, end-to-end guides for key eldercare workflows—written in clear, structured Markdown and published with MkDocs Material.

> This is a working showcase for applying modern documentation practices (docs-as-code, style gates, and AI-assisted workflows) to real-world public service tasks in Ireland.

## How this site is built

- **Docs-as-code**: Markdown in a Git repo, published with **MkDocs Material**.
- **Structure**: Content is organized by **type** (`concepts/`, `reference/`, `tasks/`) for clarity and reuse.
- **Style & quality** (coming soon): automated linting and style checks aligned to professional guides.

!!! tip
    Use the left navigation to jump between topics. Press `/` to search.
